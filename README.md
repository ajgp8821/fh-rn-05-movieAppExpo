## 1. Create Expo Project

- Run:
  - `$ npx create-expo-app -t expo-template-blank-typescript`
  - `$ cd Movie-App`
  - `$ $ npx expo start -c`

## 2. Configure Eslint

- Run:
  - `$ npm install eslint -D`
  - `$ .\node_modules\.bin\eslint --init` or `$ npm init @eslint/config`
